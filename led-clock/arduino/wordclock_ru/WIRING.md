# Word Clock RU v2 — Схема пайки и подключения

## Компоненты

| Компонент | Кол-во | Описание |
|-----------|--------|----------|
| WS2812B LED | 256 шт | Индивидуально адресуемые, 5V |
| ESP32-S3 DevKitC | 1 | Контроллер, WiFi + NTP |
| Конденсатор 1000µF 6.3V | 1 | На вход питания LED-ленты |
| Резистор 330Ω | 1 | Между GPIO48 и DATA IN первого LED |
| Блок питания 5V 5A | 1 | ~15W для 256 LED при 40 яркости |
| Провода | — | 22AWG для питания, 26AWG для данных |

## Сетка 16×16 — Serpentine (змейка)

LED нумеруются слева направо на чётных строках, справа налево на нечётных.
Это стандартная раскладка для LED-лент, уложенных зигзагом.

```
Вид СЗАДИ (сторона пайки, зеркально к лицевой):

  DATA IN (GPIO48) ──► здесь
  ┌──────────────────────────────────────────────────┐
  │                                                  │
  │  Row 0  (L→R):  LED   0 →   1 →   2 → ... →  15 │──┐
  │  Row 1  (R→L):  LED  31 ←  30 ←  29 ← ... ←  16 │←─┘
  │  Row 2  (L→R):  LED  32 →  33 →  34 → ... →  47 │──┐
  │  Row 3  (R→L):  LED  63 ←  62 ←  61 ← ... ←  48 │←─┘
  │  ...                                              │
  │  Row 14 (L→R):  LED 224 → 225 → 226 → ... → 239 │──┐
  │  Row 15 (R→L):  LED 255 ← 254 ← 253 ← ... ← 240 │←─┘
  │                                                  │
  └──────────────────────────────────────────────────┘
```

### Формула LED-индекса

```
ledIndex(row, col):
    if row % 2 == 0:  return row * 16 + col        // L→R
    else:             return row * 16 + (15 - col)  // R→L
```

## Схема подключения

```
                         ┌─────────────────┐
                         │   ESP32-S3      │
                         │   DevKitC       │
                         │                 │
  5V PSU ─── +5V ────────┤ 5V          GND ├──── GND PSU
                         │                 │
                    330Ω │ GPIO48          │
                      ├──┤                 │
                      │  └─────────────────┘
                      │
              ┌───────┴──────────────────────────────────┐
              │  WS2812B LED Strip (256 LEDs)             │
              │                                          │
  5V PSU ─────┤ +5V                                      │
  GND PSU ────┤ GND                                      │
  DATA ───────┤ DIN                                      │
              │                                          │
              │  LED 0 → 15 → 31 ← 16 → 32 → ... → 255 │
              └──────────────────────────────────────────┘
              │
              ╪ 1000µF 6.3V (между +5V и GND, у начала ленты)
              │
```

## Инжекция питания

При 256 LED на полной яркости белого — до 15A. При BRIGHTNESS=40 (~16%) хватает 5A.
Рекомендуется инжекция питания каждые 4 ряда:

```
5V/GND ──┬── LED 0   (начало, row 0)
         ├── LED 64  (начало row 4)
         ├── LED 128 (начало row 8)
         └── LED 192 (начало row 12)
```

Паять +5V и GND напрямую к контактным площадкам ленты в этих точках.

## Резка ленты

Лента WS2812B (60 LED/m) режется на полосы по 16 LED:

```
1 полоса = 16 LED = 26.7 см (при шаге 16.7мм)
16 полос = 256 LED = 4.27 м ленты

Раскладка (вид сзади):
    ┌────────────────────────┐
    │ → → → → → → → → → → →│──┐  Row 0 (LED 0-15)
    │ ← ← ← ← ← ← ← ← ← ←│←─┘  Row 1 (LED 16-31)
    │ → → → → → → → → → → →│──┐  Row 2 (LED 32-47)
    │ ← ← ← ← ← ← ← ← ← ←│←─┘  Row 3 (LED 48-63)
    │         ...             │
    │ → → → → → → → → → → →│──┐  Row 14 (LED 224-239)
    │ ← ← ← ← ← ← ← ← ← ←│←─┘  Row 15 (LED 240-255)
    └────────────────────────┘

    Соединение между полосами: 3 провода (5V, GND, DATA)
    Длина перемычки: ~5мм (только развернуть направление)
```

## Порядок сборки

1. Нарезать ленту: 16 полос по 16 LED
2. Приклеить на основу (фанера/акрил 27×27 см) зигзагом
3. Припаять перемычки DATA между полосами (3 провода: 5V, DIN→DOUT, GND)
4. Припаять инжекцию питания на rows 0, 4, 8, 12
5. Припаять 330Ω резистор на DATA IN первой полосы
6. Припаять конденсатор 1000µF между +5V/GND у начала
7. Подключить ESP32-S3: GPIO48 → резистор → DATA, 5V, GND
8. Залить прошивку, проверить startupSweep
9. Наложить трафарет с буквами (винил/лазерная резка)
